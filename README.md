# CodeAlpha Disease Prediction

## Project Overview
This project was developed as part of the CodeAlpha Machine Learning Internship.

The objective of this project is to predict whether a patient is likely to have diabetes based on medical data using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn

## Dataset
- Pima Indians Diabetes Dataset

## Machine Learning Algorithm
- Logistic Regression

## Features Used
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## Project Workflow
1. Load the dataset
2. Preprocess the data
3. Split data into training and testing sets
4. Train the Logistic Regression model
5. Evaluate model performance
6. Predict disease outcomes

## Results
- Disease Prediction Accuracy: **74.68%**

## How to Run
1. Install required libraries:
   ```bash
   pip install pandas numpy scikit-learn
   ```

2. Run the program:
   ```bash
   python disease_prediction.py
   ```
